노드

탭: 노드 / 편집 / 미리보기 / 아이템

    편집

    노드 선택 → 제목/내용/선택지/규칙 편집
    선택지
    선택지 클릭 시: (혼란 변화 → 아이템 효과 적용 → 다음 노드 이동) ※ 플레이어 화면에는 혼란 수치가 표시되지 않고, 분위기만 미세하게 달라짐

    미리보기

    플레이 모드: 선택지 클릭 → (혼란/아이템) → 다음 노드
    인벤토리
    아이템은 인스턴스(개별) 단위로 존재하며, 선택지에서 사용(uses 감소)돼.
    시작
    왼쪽에서 시작 노드를 선택하거나, 노드 0001부터 시작해.

    아이템

    아이템 정의(타입) 생성/편집: maxUses=숫자 또는 무제한
    아이템 목록
    선택 아이템 편집
    기본 속성
    qty/dur 범위는 현재 0~10으로 클램프돼.
    아이템 상태 (최대 3)
    base (기본)
    dur0 (파손/0)
    special (특수)
    special.nodeId는 문자열로 저장돼. 필요하면 JS에서 숫자 패딩 처리 가능.
    itemId는 중복되면 안 돼.
