게임만들기

노드

탭: 노드 / 편집 / 미리보기 / 아이템

    편집

    노드 선택 → 제목/내용/선택지/규칙 편집
    Node
    제목 / 본문
    Choices
    텍스트 / 이동 / 조건 / 아이템
    선택지
    선택지는 ‘미리보기’에서 클릭하면 효과(아이템 획득/사용)가 적용되고 다음 노드로 이동해.

    미리보기

    플레이 모드: 선택지 클릭 → 조건 검사 → 효과 적용 → 다음 노드
    인벤토리
    아이템은 인스턴스 단위(개별)로 존재하며, 선택지에서 사용(uses 감소)돼.
    시작
    왼쪽에서 시작 노드를 선택하거나, 노드 1부터 시작해.

    아이템

    아이템 정의(타입) 생성/편집: maxUses=숫자 또는 무제한
    아이템 목록
    선택 아이템 편집
    itemId는 중복되면 안 돼. (예: key, knife, coin 같은 짧은 키)
